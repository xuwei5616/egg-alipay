# egg-alipay

[官方文档](https://www.yuque.com/chenqiu/alipay-node-sdk)

## 安装

```bash
npm i @axolo/egg-alipay --save
```

## 开启

```js
// config/plugin.js
exports.alipay = {
  enable: true,
  package: '@axolo/egg-alipay',
};
```

## 配置

```js
// {app_root}/config/config.default.js
exports.alipay = {
};
```

请到 [config/config.default.js](config/config.default.js) 查看详细配置项说明。
